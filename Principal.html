Saber mas sobre TICKET-DAJE

El desarrollar un prototipo de un servicio web, donde el usuario pueda comprar boletos de

manera sencilla, también es el implementar una base de datos sobre donde se registren las compras, datos del usuario.


Funcionamiento

• Se puede realizar un trámite de compra.


•Se puede hacer transacciones de movimiento en el servicio web a otra sección.


•Mas facil la compra del boleto.


Ejemplo del boleto

Se podrias decir que tendra forma de un boleto de entrada donde vendra el usuario y el nombre del trasporte a tomar.
